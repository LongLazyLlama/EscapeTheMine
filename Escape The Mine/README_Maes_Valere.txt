Info about the Game
================================
Menu Controls
- Enter => Start Game
- LMB   => To click on the controls or highscores button

InGame Controls:
- space => Jump
- LMB   => Throw pickaxe
- Backspace => Suicide button

GameOverScreen Controls:
- Enter => Retry
- LMB   => To click on the retry or menu button

Special Controls InGame:
- b a t => Transform the player into a bat
